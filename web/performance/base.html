# 输入 http://www.baidu.com 后浏览器做了什么

  1. 用户通过浏览器发送一个 http 的请求到指定的主机
  2. 服务器接收到该请求，对该请求进行分析和处理
  3. 服务器处理完成以后，返回对应的数据到用户机器
  4. 浏览器接收服务器返回的数据，并根据接收到的进行分析和处理

# 客户端 -> 服务器

客户端发送一个 http 请求到指定的服务器 -> 服务端接收并处理请求 -> 返回数据到客户端
